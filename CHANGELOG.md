# Changelog

All notable changes to this project will be documented in this file.  This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html) for the code and uses a date‑based versioning scheme for the data.

## v2025.11 – 2025‑11‑12

* Initial release of the narrative fiscal consolidation dataset for 14 Sub‑Saharan African countries.
* Includes annual series of tax, spending, and total consolidation actions from 1990 onward.
* Provides data in CSV and Stata `.dta` formats.
* Adds a working paper appendix containing the narrative text excerpts used to identify episodes.
* Sets up the documentation site with variable definitions, methodological notes, and a country coverage table.
